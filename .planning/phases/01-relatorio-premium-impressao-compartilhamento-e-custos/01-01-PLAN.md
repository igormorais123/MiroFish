## Objetivo

Adicionar toolbar profissional ao relatório finalizado com funcionalidades de impressão formatada (preservando estética INTEIA com tipografia Cormorant + Inter, paleta dourada #c9952a, layout A4), compartilhamento social (WhatsApp, Email, Copiar Link) e calculadora de custos estimados por etapa do pipeline de análise.

**Propósito:** Permitir que usuários finalizem a jornada de análise exportando relatórios de forma profissional, compartilhando insights e visualizando o custo computacional da análise.

**Output:** Step4Report.vue com toolbar, painel de custos, CSS de impressão integrados. Frontend-only Phase 1; backend real tracking em Phase 3.

1. No `<script setup>`, adicione as definições de preços conhecidos e a função para calcular custos:

```javascript
// Model pricing (USD per 1K tokens)
const MODEL_PRICING = {
  'sonnet': 0.003,      // claude-sonnet-4-6
  'opus': 0.015,        // claude-opus-4-6
  'bestfree': 0,        // BestFREE
  'deepsearch': 0.001   // estimate
};

// Estimativas de tokens por etapa (baseado em típicos)
const PIPELINE_ESTIMATES = {
  'GraphRAG': {
    model: 'bestfree',    // BestFREE para volume alto
    inputTokens: 8000,    // ~8K para processamento de chunks
    outputTokens: 2000    // ~2K resumo/embedding
  },
  'Perfis': {
    model: 'sonnet',      // Sonnet para geração rápida
    inputTokens: 12000,   // ~12K contexto + agentes
    outputTokens: 3000    // ~3K profiles
  },
  'Simulação': {
    model: 'bestfree',    // BestFREE para multi-turn
    inputTokens: 25000,   // ~25K turnos de simulação
    outputTokens: 10000   // ~10K saídas
  },
  'Relatório ReACT': {
    model: 'sonnet',      // Sonnet para qualidade
    inputTokens: 15000,   // ~15K contexto + ferramentas
    outputTokens: 5000    // ~5K relatório
  },
  'Helena Strategos': {
    model: 'opus',        // Opus para análise estratégica
    inputTokens: 20000,   // ~20K histórico + contexto
    outputTokens: 8000    // ~8K análise
  }
};

const pipelineCosts = computed(() => {
  const breakdown = [];
  let totalTokens = 0;
  let totalCost = 0;

  for (const [stageName, estimate] of Object.entries(PIPELINE_ESTIMATES)) {
    const totalInputOutput = estimate.inputTokens + estimate.outputTokens;
    const pricing = MODEL_PRICING[estimate.model];
    const stageCost = (totalInputOutput / 1000) * pricing;

    totalTokens += totalInputOutput;
    totalCost += stageCost;

    breakdown.push({
      name: stageName,
      model: estimate.model,
      tokens: totalInputOutput,
      cost: stageCost
    });
  }

  return {
    stages: breakdown,
    totalTokens,
    totalCost
  };
});
```

## Modelo de Ameaças

### Limites de Confiança

| Limite | Descrição |
|--------|-----------|
| Frontend → Clipboard | Script pode ler/escrever na área de transferência (Clipboard API) |
| Frontend → Window.open | Script pode abrir janelas/abas (print, whatsapp.com, wa.me) |
| Frontend → LocalStorage | Script pode persistir dados no navegador |

### Registro de Ameaças STRIDE

| ID | Categoria | Componente | Disposição | Plano de Mitigação |
|-----|-----------|-----------|-----------|-----------------|
| T-01-001 | Information Disclosure | Clipboard (Copiar Link) | accept | URL é pública (já está na barra de endereço). Sem risco de exposição de PII. |
| T-01-002 | Spoofing | Window.open para WhatsApp/Email | accept | Invoca APIs oficiais (wa.me, mailto:). Sem risco de XSS ou phishing. |
| T-01-003 | Tampering | Print CSS | accept | CSS é local, aplicado apenas na janela de impressão. Sem impacto no estado da aplicação. |
| T-01-004 | Denial of Service | Múltiplos cliques em Imprimir | accept | Abrir múltiplas janelas de print é incômodo mas não causa indisponibilidade. Usuário pode fechar. |
| T-01-005 | Information Disclosure | Cost Panel | accept | Custos são estimativas públicas baseadas em pricing de modelos conhecidos (Sonnet $0.003/1K, Opus $0.015/1K). Sem exposição de métricas reais até Phase 3. |

**Nenhuma ameaça classificada para mitigar nesta phase. Todos os riscos são baixos e aceitáveis para frontend-only Phase 1.**
